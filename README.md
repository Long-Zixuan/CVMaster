<h1 align="center">Paste Master</h1>>
这个软件是利用python的库来模拟键盘输入以达到类似粘贴的效果，这样我们就可以在不允许粘贴的网站粘贴我们的文本了

This software uses Python libraries to simulate keyboard input and achieve a paste like effect, so that we can paste our text on websites that do not allow pasting
